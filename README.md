# Gold Recovery Optimization Project

## Overview
This project analyzes industrial mining data to predict and optimize gold recovery rates during the ore purification process.

## Objective
- Validate recovery calculations from raw data
- Analyze metal concentration across purification stages
- Build machine learning models to predict recovery rates
- Optimize process efficiency using data-driven insights

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Time Series Analysis

## Methodology
- Performed data validation and preprocessing on time-indexed datasets
- Verified recovery calculations with near-zero MAE
- Analyzed metal concentration trends (Au, Ag, Pb) across stages
- Removed outliers using statistical methods (IQR)
- Built and evaluated machine learning models (Random Forest, Gradient Boosting)
- Used time-series cross-validation for model evaluation

## Key Results
- Achieved **9.19% SMAPE** using Random Forest model
- Identified key process variables impacting gold recovery
- Confirmed rougher stage as the most influential stage in the process

## Business Impact
- Provided predictive insights to improve gold recovery efficiency
- Identified optimization opportunities in the purification process
- Demonstrated real-world application of machine learning in industrial operations
